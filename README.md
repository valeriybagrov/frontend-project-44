### For installing brain-games, you should copy and paste these commands in your bash terminal:
``` sh
make install
```
``` sh
npm link
```

### To start the game, type a specific name from the suggested below:
`brain-games`
`brain-even`
`brain-calc`
`brain-gcd`
`brain-progression`
`brain-prime`
